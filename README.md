# Spext Music UI

[Visit Demo](https://spext-music-ui.web.app)

### To Run

```bash
git clone https://github.com/Anikeshpatel/spext_music_ui.git
cd spext_music_ui
```

```bash
npm install
```

```bash
npm start
```
